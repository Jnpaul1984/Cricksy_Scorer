## 1. Tracker Purpose

This file records the actual learning journey.

It should be used to prevent drift, avoid repeating topics unnecessarily, and provide evidence before marking lessons, exercises, projects, or assessments complete in Notion.

This file tracks:

- class sessions
- curriculum location
- lesson objectives
- exercises completed
- assessment results
- grades
- strengths
- weaknesses
- teacher feedback
- next session focus
- Notion update recommendations

---

## 2. Session Logging Rules

- Add one entry per completed class session.
- Do not mark a lesson complete unless the class session evidence supports it.
- Do not mark a project complete unless deliverables were produced or reviewed.
- Do not advance to the next module until the current module completion gates are reviewed.
- If a topic expands beyond the current lesson scope, record it as a future note instead of changing the current session path.
- Notion should be updated after each confirmed session, but this tracker remains the source-controlled evidence log.
